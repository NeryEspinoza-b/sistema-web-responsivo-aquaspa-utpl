# Sistema Web Responsivo para la Gestión de un Establecimiento del Sector SPA

Este repositorio contiene el desarrollo del sistema web responsivo implementado como parte del Trabajo de Integración Curricular (TIC), cuyo objetivo es optimizar los procesos de gestión de un establecimiento del sector SPA, permitiendo la automatización y organización de la información administrativa y operativa.

El sistema web responsivo permite gestionar de manera eficiente los principales procesos del establecimiento SPA, tales como el registro y administración de clientes, la gestión de servicios, la reserva de citas, el manejo del historial clínico, así como la generación de reportes y estadísticas para la toma de decisiones administrativas. La solución fue desarrollada bajo una arquitectura cliente-servidor, garantizando la separación de responsabilidades entre la interfaz de usuario y la lógica del servidor, y aplicando la metodología ágil SCRUM para organizar el proceso de desarrollo.

## Tecnologías utilizadas

Frontend: React.js, Vite, Tailwind CSS, JavaScript, HTML5 y CSS3.  
Backend: Node.js, Express.js, MongoDB, Mongoose, JSON Web Tokens (JWT), Zod y Nodemailer.

## Configuración del entorno

Por motivos de seguridad, las variables de entorno no se incluyen en el repositorio. El archivo `.env` debe ser creado localmente por el usuario, tomando como referencia el archivo `backend/.env.example`.

## Ejecución del proyecto

Backend:  
cd backend  
npm install  
npm run dev  

Frontend:  
cd frontend  
npm install  
npm run dev
